# 🗓️ Tempus: Gestor de Tareas en Java

Tempus es una aplicación de escritorio escrita en Java que permite a los usuarios crear, organizar y gestionar sus tareas de manera eficiente. Diseñado con principios de orientación a objetos y buenas prácticas de desarrollo, es ideal como base para futuras mejoras o integración con interfaces gráficas y APIs.

## 🚀 Características

- 📋 Crear, editar y eliminar tareas
- ⏰ Agregar fechas límite y prioridades
- 🔄 Marcar tareas como completadas
- 💾 Persistencia local de datos
- 🧪 Estructura modular y lista para pruebas unitarias

## 🛠️ Tecnologías utilizadas

- **Java 22**
- **Maven** para gestión de dependencias
- **JUnit 5** (opcional para testing)
- Posibilidad de expandirse con **Spring Boot** o GUI (JavaFX, Swing)
